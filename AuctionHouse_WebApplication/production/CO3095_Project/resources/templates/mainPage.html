Welcome to Auctionhouse website

This application is for bidding houses.

Most features require logging in. Not authorized users have access to the welcome screen and login or registration panel.

Admin (seller) can:

  • Create auction and submit minimum bidding amount
  • View list of all users with possibility to delete user
  • View list of all auctions with editing or deleting auction

Common user (buyer) can:

  • Submit maximum amount of bid.
  • View list of all auctions wirth only bidding allowed.

Every authorized user can:

  • View his own profile
  • View auctions
